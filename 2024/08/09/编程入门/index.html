什么是编程？

1.关于编程，编程语言，编译器与编程环境

1.1 编程的概念

我们常说的编程是指通过计算机使用编程语言（如 C，C++，Java，Python等）编写脚本，让计算机按照编程者的思维去执行一系列指令。
通过编程，我们可以创建软件、应用程序、网站、游戏等。简单来说，编程就是用计算机能够理解的语言来编写指令，让计算机执行特定的任务。

  • 下面是一个由C++编写的简单的程序： 这个程序的作用就是在命令行界面上打印出 Hello World! 语句。

1.2 编程语言

编程语言是人和计算机之间沟通的工具，是一种特殊的语言，用于编写程序。编程语言有很多种，每种都有自己的特点和用途。常见的编程语言有： C，C++，Java， Python，我们大一入门所学习的第一门编程语言是 C++ 。

1.3 编译器

什么是编译器？这是一个好问题
所谓编译器，就是把你的编写的程序（如上述Hello World.cpp），给翻译成计算机可以直接运行的二进制文件，让它理解
过程可以抽象表示为：

  • 编程语言（程序） ——> 编译器 ——> 二进制文件 ——> 计算机

其中编译器起到了一个桥梁的作用，因此在下载好编译器后还需要将它的路径设置到计算机的环境变量中，并且与编辑器进行连接。

1.4 编程环境

编程环境是指程序员用来编写、测试和调试代码的所有工具和资源的集合，包括以下部分：

  • 编辑器/IDE（集成开发环境）：用于编写代码的软件
  • 编译器：用于将代码转化为可执行形式的工具。

1.4.1 文本编辑器

文本编辑器通常提供基本的文本编辑功能，例如语法高亮、自动缩进、括号匹配等，不仅仅是编程，还适用于多种用途，轻量级且灵活。

常见的编辑器有：Visual Studio Code (VS Code)，这是一款微软推出的免费编辑器，支持多种语言，并且有大量的扩展。

1.4.2 IDE（集成开发环境）

IDE 是为软件开发而设计的一套完整的编程软件，它自带了编写、调试、构建、运行代码一整套流程的所有功能。IDE 通常包含一个编辑器以及许多其他功能，以简化整个开发过程。
Visual Studio 是C++的常用的IDE。作为一个完整的 IDE，Visual Studio 比它更轻、更便携的对应物 VS Code 更适合更复杂的系统。
Visual Studio 2022 是 Visual Studio 目前发布的最新版本，其社区版，即Visual Studio 2022 Community是免费的。

1.4.3 C++编辑器的选择

目前常见的C++编辑器有3种，它们分别是：Visual Studio，VS Code和Dev C++，其中Dev C++虽然免费，但是由于功能有限且版本多年未更新，严重落后于时代，所以不推荐使用，下面只讨论Visual Studio与VS Code的优劣。

Visual Studio具有以下优缺点:

优点：

  1. 集成度高：Visual Studio 提供了完整的开发环境，包括编译器、调试器、项目管理工具等。
  2. 功能强大：Visual Studio 的调试器很强大，支持断点、变量观察、调用堆栈查看等功能，非常适合复杂项目的调试。
  3. 配置简单：Visual Studio 下载时会自带编译器，不用额外配置。

缺点：

  1. 体积较大：Visual Studio 安装包体积较大，安装过程可能比较耗时。
  2. 资源消耗较高：由于功能全面，Visual Studio 在运行时对系统的资源消耗相对较高。

VS Code具有以下优缺点:

优点：

  1. 轻量级：VS Code 启动速度快，占用系统资源较少。
  2. 跨平台：VS Code 可以在 Windows、macOS 和 Linux 上运行。
  3. 社区活跃，插件丰富：拥有庞大的开发者社区和丰富的插件资源，支持大量插件，用户可以根据自己的需求定制开发环境。

缺点：

  1. 调试功能不如 Visual Studio：虽然 VS Code 也提供了调试功能，但在某些方面可能不如 Visual Studio 强大。
  2. 配置复杂：为了获得类似 IDE 的功能，用户需要安装多个插件并进行一些额外的配置，并且需要手动配置环境，对新手不太友好。

综上，由于 Visual Studio 的环境配置较为简单，功能强大，比较适合新手入门学习编程，因此我推荐 Visual Studio 作为C++的入门编辑器，到后面对编辑器以及环境配置有一定了解后在选择其他代码编辑器也不迟

  • 以上纯属个人观点，如有错误，欢迎指正。
